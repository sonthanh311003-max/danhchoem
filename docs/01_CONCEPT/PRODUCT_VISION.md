# The Memory Archive - Product Vision v2.0
# Core Constitution

This document defines the emotional and visual identity of MemoryOS. It takes precedence over all other requirements and code configurations.

---

## 🏛️ Fictional World: The Memory Archive
MemoryOS is not software; it is a digital destination. It is a hidden, ancient library preserving humanity's happiest moments: promises, weddings, first kisses, and peaceful goodbyes.

## 💌 The Keeper: Emma
Emma is the Keeper of Memories. She is not a chatbot or an AI. She is a gentle guardian. She speaks softly, reassures, and protects.
* **Greeting**: *"Welcome. You've brought something precious. Let's preserve it together."*
* **Ritual**: *"It will be safe here."*

## 📜 Storytelling Lexicon (Terminology)
* **Create Project** ──> **Preserve a Memory**
* **Upload Photos** ──> **Bring the moments you'd like to remember**
* **Next** ──> **Continue the Story**
* **Finish / Save** ──> **Preserve Forever**
* **Page** ──> **Scene** (Opening Scene, Letter Scene, Gallery Scene, Ending Scene)

## 🕯️ Memory Ritual
When the user clicks "Preserve Forever":
1. Pause & Silence.
2. The letter gently folds back.
3. Golden dust particles rise.
4. Emma whispers: *"It will be safe here."*
5. Only then is the memory saved and transition allowed.
