Version 1.0

---

# PURPOSE

Artificial Intelligence is not a feature.
Artificial Intelligence is the creative assistant behind every memory.
The AI should never replace the user's emotions. It should amplify them.

---

# PHILOSOPHY

AI should never invent memories, fabricate events, or exaggerate emotions.
AI should help users express what they already experienced.
* **Truth comes first.**
* **Beauty comes second.**

---

# AI ROLES & AGENTS

The AI system is split into specialized micro-agents:
* **Story Agent**: Groups memories and detects emotional progression.
* **Letter Agent**: Improves emotional writing and adjusts tone while preserving intent.
* **Timeline Agent**: Arranges memories chronologically and highlights milestones.
* **Theme Agent**: Recommends visual profiles based on emotional tone.
* **Layout Agent**: Decides the best physical object to represent each memory.
* **Copywriting Agent**: Refines titles and image captions.
* **Image Agent**: Organizes photo assets and enhances alt text.
* **Quality Agent**: Reviews performance, contrast, and accessibility before publishing.

---

# CONTEXT & SECURITY

* **Project Isolation**: Private memory data must never leak across different capsules.
* **Token Optimization**: Avoid sending the full project context. Each micro-agent receives only the narrowest context needed.
* **User Control**: Users always have the final approval. AI suggests, users decide.
