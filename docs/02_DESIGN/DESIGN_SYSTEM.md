# DIGITAL MEMORIES
# DESIGN SYSTEM
Version 1.0

---

# PURPOSE

This document defines the visual language of Digital Memories.
Every UI element must follow these rules. Never invent styles randomly.

---

# DESIGN PHILOSOPHY

Digital Memories should feel like a handwritten love letter, a luxury coffee table book, or a handcrafted keepsake — NOT a SaaS dashboard or a generic website.

---

# COLOR & TYPOGRAPHY SYSTEM

* **Primary Colors**: Warm Ivory, Soft Paper, Cream White. No pure white.
* **Accent Colors**: Dusty Rose (`#E96A87`), Deep Burgundy (`#6B1D2F`), Muted Gold (`#D9B36A`).
* **Typography**: Elegant display Serif (`Cormorant Garamond`) and clean readable Sans-serif (`Inter`). Spacing between lines should feel generous and breathe.
