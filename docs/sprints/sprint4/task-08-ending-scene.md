# Ending Scene

Priority: High
Estimated Time: 3-5 hours

---

## Goal
Design and build the emotional Ending Scene with starscapes and post-keepsake calls to action.

---

## Ending Pipeline
`Music Fades -> Sky Darkens (Stars appear) -> Thank You Message -> CTAs (Replay / Share / Create)`

---

## Technical Features
* Smooth HTML5 Canvas starfield rendering or CSS particles background.
* Delayed fade-in for action CTAs to let the user sit with the emotion.
* Share dialog integration (Web Share API fallback).
* Call-to-action button to navigate back to registration flow.
