# Transition Engine

Priority: Critical
Estimated Time: 5-8 hours

---

## Goal
Design and build the Transition Engine for physical-object-based cinematic page shifts.

---

## Transition Presets Registry
* **Envelope (Open Animation)**: Wax seal break -> Envelope lid opening (rotateX) -> Letter slide-up.
* **Book (Page Turn)**: Spine shadow -> rotateY flipping transition.
* **Gallery (Photo Scatter)**: Staggered translation & random rotations (Polaroid slide-in effect).
* **Timeline (Horizontal Reveal)**: Mask clipping path slide -> opacity reveal cascade.
* **Ending (Fade to Stars)**: Ambient sky brightness decrease -> constellation draw.

---

## Technical Features
* Transition trigger controls mapped to individual Scene Types.
* Frame Motion orchestrator logic.
* Audio crossfading transition (seamless track shift).
