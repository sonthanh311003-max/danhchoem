# Sprint 1
# Task 01
# Landing Experience

Priority: Critical
Estimated Time: 4-6 hours

---

## Goal
Build a premium landing page.
The landing page should feel emotional.
Users should immediately understand: "This is different."
This is NOT another website builder.
This is an experience builder.

---

## Tech Stack
* Next.js
* TypeScript
* TailwindCSS
* Framer Motion
* Lenis
* GSAP (optional)
* Lucide

---

## Sections
Landing contains only:
* Hero
* Experience Preview
* Features
* Templates
* FAQ
* Footer
Nothing else.

---

# HERO
The hero fills 100vh.
Center aligned.
The background is extremely minimal:
* Soft paper texture.
* Very subtle animated grain.
* Warm lighting.

## Hero Animation
When page loads:
* **0.0s**: Fade in.
* **0.4s**: Envelope appears.
* **1.0s**: Envelope slowly opens.
* **1.8s**: Letter slides upward.
* **2.4s**: Headline fades in.
* **3.0s**: CTA appears.
Entire animation: 4 seconds. Must remain skippable.

## Headline
Every memory deserves a beautiful home.
Use elegant typography. Max width 700px. Center aligned.

## Subheadline
Create beautiful interactive memory experiences for the people you love.

## CTA
One primary button only: **Create My Memory**. No secondary CTA.

---

# EXPERIENCE PREVIEW
Horizontal cards. Each card previews one experience:
* Letter
* Book
* Timeline
* Film Roll
* Memory Box

Hover interaction: lift, shadow, slight rotation, soft glow.

---

# FEATURES
Three columns:
* Emotion First
* Beautiful by Default
* AI Assisted

Each feature uses an illustration. No stock icons.

---

# TEMPLATE SECTION
Large cards: Couple, Wedding, Travel, Birthday, Family.
Each card includes: Preview, Description, CTA.

---

# FAQ
Accordion. Smooth animation. No layout shift.

---

# FOOTER
Minimal: Logo, Navigation, Social, Copyright.

---

# Motion
Never use generic animations. Everything should feel soft. Spring based. Natural. No bounce.
