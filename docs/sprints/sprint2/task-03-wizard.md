# Sprint 2
# Task 03
# Experience Wizard

Priority: Critical
Estimated Time: 5-8 hours

---

## Goal
Create a multi-step Experience Wizard.

## Requirements
* One question per screen.
* Animated transitions.
* Progress indicator.
* Autosave after every step.
* User can go back.
* User never loses progress.
* Responsive.
* Use Framer Motion.
* Do not implement AI yet. Mock generation for now.

---

## Steps
1. **Who is this memory for?** (My Partner, Family, Friend, Myself)
2. **What are we celebrating?** (Birthday, Anniversary, Proposal, Wedding, Trip, Graduation, Other)
3. **Project Name**
4. **Choose a Theme** (Classic, Luxury, Vintage, Minimal, Autumn)
5. **Upload Photos** (Drag & Drop, Multiple Upload, Preview Grid)
6. **Do you want to add music?** (Skip, Upload, Choose Later)
7. **Would you like AI to help write your letter?** (Yes, No)
8. **Generating...** (AI is preparing your experience...)
