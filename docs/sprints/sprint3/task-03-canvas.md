# Center Canvas

Priority: Critical
Estimated Time: 4-6 hours

---

## Goal
Design and build the Center Canvas workspace for rendering keepsake objects.

## Core Rules
* The canvas must always stay perfectly centered within the parent container.
* Content must never crop or overflow.
* Recalculate dimensions dynamically using ResizeObserver.

---

## Features
* Aspect ratio lock support.
* Zoom scale controls: `50%`, `75%`, `100%`, `125%`, `Fit Width`, `Fit Screen`.
* Smooth CSS transition scales.
* No layout shift on resize.
