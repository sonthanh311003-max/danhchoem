Priority: High

## Goal
Design and build the Gallery Editor for managing, sorting, and captioning memory photos.

---

## Features
* Drag-and-drop grid reordering.
* Cover photo selector.
* Photo captions editing.
* Image compression before uploading.
* Lazy loading images.
* Smooth grid transition animations (Framer Motion).
* Keyboard navigation for accessibility.
