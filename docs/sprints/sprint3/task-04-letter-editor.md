# Sprint 3
# Task 04
# Letter Editor

Priority: Critical
Estimated Time: 5-8 hours

---

## Goal
Design and build the interactive Letter Editor experience.

## Visual Design
* Warm paper texture background.
* Multi-layered realistic paper shadow.
* Typography optimized for reading (60-70 characters max width).
* Viewport-locked, internal scrolling only.

---

## Technical Features
* Auto save with debounce.
* Markdown formatting support.
* Undo/Redo typing state.
* Live counters: character count, word count, estimated reading time.
* Placeholder support.
* Reduced Motion preference toggle.

---

## Deliverables
Before coding:
1. Explain editor layout & typography spacing.
2. Explain markdown editing implementation strategy.
3. Explain live counters calculation math.

After approval:
* Implement.
